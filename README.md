### 自行部署
**此项目使用了openai python sdk，所以可以搭配[one-api](https://github.com/songquanpeng/one-api)使用**
```bash
git clone https://github.com/hqycloud/GPT-for-yhchat.git
pip install -r requirements.txt
```
**打开main.py填写配置**
```bash
python main.py
```

**项目内的注释已使用Ai补全！**
